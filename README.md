# Kiija Math Wizard - A Learning Tool
An infinite math problem set generator for fundametal math topics.

Practice makes perfect.

* Addition
* Subtraction
* Multiplication (todo)
* Division (todo)

### Deploying backend
1. cd math-drills/mathtime
2. ./gradlew build
3. java -jar ./build/libs/mathtime-0.0.1-SNAPSHOT.jar
